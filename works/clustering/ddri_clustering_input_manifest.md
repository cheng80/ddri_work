# Ddri Clustering Input Manifest

작성일: 2026-03-11
목적: 군집화 1차 작업에 사용할 입력 데이터 목록을 고정한다.

## 1. 사용 범위

- 분석 대상: 강남구 따릉이 대여소
- 학습 기준 기간: 2023-01-01 ~ 2024-12-31
- 테스트 기준 기간: 2025-01-01 ~ 2025-12-31
- 1차 목표: 대여소 이용 패턴 기반 군집화

## 2. 1차 군집화 필수 입력

### 2.1 따릉이 이용 이력

학습용:
- `3조 공유폴더/2023 강남구 따릉이 이용정보/*.csv`
- `3조 공유폴더/2024 강남구 따릉이 이용정보/*.csv`

테스트 분리 보관용:
- `3조 공유폴더/2025 강남구 따릉이 이용정보/*.csv`

주요 컬럼:
- `대여일시`
- `대여 대여소번호`
- `반납일시`
- `반납대여소번호`
- `이용시간(분)`
- `이용거리(M)`

### 2.2 강남구 대여소 기준 정보

후보 파일:
- `3조 공유폴더/강남구 대여소 정보 (2023~2025)/2023_강남구_대여소.csv`
- `3조 공유폴더/강남구 대여소 정보 (2023~2025)/2024_강남구_대여소.csv`
- `3조 공유폴더/강남구 대여소 정보 (2023~2025)/2025_강남구_대여소.csv`

주요 컬럼:
- `대여소번호`
- `대여소명`
- `위도`
- `경도`
- `자치구`

## 4. 연도별 대여소 마스터 현황

실제 확인 결과:

- 2023년 대여소 수: 175
- 2024년 대여소 수: 176
- 2025년 대여소 수: 179
- 2023~2025 공통 대여소 수: 169

연도별 단독 대여소:

- 2023 전용: `2343`, `2393`, `2412`, `2428`
- 2024 전용: `4933`
- 2025 전용: `4937`, `4938`, `4939`, `4940`, `4941`, `4942`

## 5. 확정 기준

군집화 1차 산출물은 아래 기준으로 확정한다.

- 기준 대여소 집합: 2023~2025 공통 169개 대여소

## 6. 확정 이유

아래 이유로 `공통 169개 대여소` 기준이 1차 분석과 ML 베이스라인에 가장 적절하다.

- 학습 구간과 테스트 구간의 대여소 집합을 동일하게 유지할 수 있음
- 신규 개설/폐쇄 대여소로 인한 구조 변화를 1차 분석에서 제거할 수 있음
- 군집화 결과 해석이 단순해짐
- 일반적인 예측/패널 분석 기준에서 안정적인 balanced panel에 가까운 형태를 만들 수 있음

## 7. 후속 확장 과제

2025년에만 존재하는 신규 대여소는 1차 군집화 기준에서는 제외한다.

후속 과제로 별도 분리:

- 신규 대여소 cold-start 예측
- 신규 대여소 환경 기반 수요 추정
- 기존 대여소와 신규 대여소를 분리한 이중 모델 전략

## 8. 연도별 스테이션 증감 처리 방침

앞으로 프로젝트 전반에서 아래 방침을 기본 원칙으로 적용한다.

### 1차 분석/군집화/ML 베이스라인

- 학습 이력이 존재하는 `기존 운영 스테이션`만 메인 분석 대상에 포함
- 공통 운영 스테이션 기준을 우선 사용
- 신규 스테이션은 메인 평가셋에 강제로 포함하지 않음

### 신규 스테이션

- `cold-start` 문제로 별도 분리
- 본 모델의 메인 성능 평가에서는 제외 가능
- 후속 확장 과제로 환경 기반 추정 또는 별도 모델 설계

### 소멸 스테이션

- 테스트 시점에 존재하지 않으면 예측 대상에서 제외
- 운영 변화 이슈로 별도 기록

### 현재 프로젝트 적용 예시

- 테스트셋에서 학습 구간 군집 라벨이 없는 스테이션은 메인 평가셋에서 제외
- 예외 건은 별도 목록으로 관리하고 레포트에 사유를 기록
