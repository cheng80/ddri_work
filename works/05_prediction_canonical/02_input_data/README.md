# Input Data Guide

이 폴더는 `canonical_data` 생성에 사용된 입력 범위를 설명하는 안내 경로다.

정본 생성 기준 입력:

- 대표 15개 원천 시계열
- 전체 161개 원천 시계열
- `rep15` warm-up 계산용 `2022-12` 대여이력

현재 공유폴더에는 최종 `canonical_data`만 남겨 두었기 때문에, 원천 입력 스냅샷은 이 경로에 보관되어 있지 않다.
