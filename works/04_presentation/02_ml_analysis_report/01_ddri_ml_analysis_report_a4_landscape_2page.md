# 1장: 개요·문제·데이터·피처·군집·모델·실험 결과

<div class="two-col-page">
<div class="main-col">

### 목적·범위

강남구 따릉이 대여소별 **시간대 수요 예측** 실험 정리. **군집화 → 대표 15개 탐색 → 161개 운영 기준선** 흐름.

- **예측 단위**: 대여소·시간 (특정 대여소, 특정 시각의 대여량)
- **활용**: 잔여 자전거 예측, 재배치 우선순위
- **평가**: 오차(RMSE), 평균오차(MAE), 설명력(R²)

### 데이터

| 구분 | 대표 15개 | 전체 161개 |
|------|-----------|------------|
| 역할 | 군집별 피처 탐색 | 운영 기준선 선택 |
| 학습/테스트 | 2023~2024 / 2025 | 2023~2024 / 2025 |

입력: 대여 이력, 대여소 정보, 날씨(기온·습도·강수·풍속), 정적 입지(지하철·버스·표고·상권), 군집

### 피처·군집

- **시간**: 출퇴근·야간·주말·공휴일전날
- **날씨**: 기온, 습도, 강수, 강한 비 여부
- **입지**: 지하철 거리, 300m 버스정류장 수, 표고, 음식점·카페 수
- **상호작용**: 비×출퇴근, 비×야간, 강수×출퇴근, 강수×야간 (날씨 단독보다 유효)
- **군집 5개**: 업무/상업 혼합(49), 아침 도착 업무(3), 주거 도착(32), 생활·상권 혼합(61), 외곽 주거(19)

### 모델·실험

- **모델**: LightGBM, CatBoost / **학습 목표**: 오차 최소화(RMSE), 건수형(Poisson)
- **실험 축**: 군집별 축소 피처, 정적 피처 확장, 날씨×시간대 상호작용, 군집별 분기(routing)

### 대표 15개 핵심 결과

|------|-------------|------|
| 주거 도착형 최선 | 0.799 | 출퇴근·교통 축소 피처 + Poisson |
| 15개 기준선 | 0.920 | 정적 피처 기준 |

## 강남구 전체 161개 운영 모델

| 비교안 | 테스트 오차 | 비고 |
|--------|-------------|------|
| **정적+날씨×시간대 상호작용** | **0.8604** | **최종 채택** |
| 군집별 정적 강조 | 0.8615 | 소폭 개선 |
| 정적 피처만 | 0.8620 | 정적 복원 효과 |
| 원본 기준선 | 0.8624 | |
| 군집별 분기 | 0.867~0.868 | 일반화 실패, 미채택 |

## 오류·날씨 상호작용

- **오류**: 저녁·아침 피크에 집중 → 날씨×시간대 상호작용이 유효
- **가중치**: 단순·월별 가중치는 개선 미미, 상호작용이 더 유효
- **군집별 분기**: 검증 개선 있었으나 테스트에서 악화 → 단일 모델 채택

## 최종 채택안

| 항목 | 내용 |
|------|------|
| 모델 | LightGBM |
| 데이터 | 강남구 전체 161개 |
| 피처 | 시간 + 날씨 + 정적 입지 + **날씨×시간대 상호작용** |
| 테스트 오차 | 0.8604 |
| 미채택 | 군집별 분기, 단순 가중치 |

## 결론

1. 군집화·대표 15개 실험은 대여소 공간 역할 해석과 피처 발굴에 유효했다.
2. 운영 기준은 **161개 단일 모델**이 군집별 분기보다 안정적이었다.
3. 최선 조합: 정적 피처 + **날씨×시간대 상호작용** (비·강수×출퇴근·야간).
4. 다음: 예측값 → 잔여 자전거·부족 위험 후처리 규칙 설계
