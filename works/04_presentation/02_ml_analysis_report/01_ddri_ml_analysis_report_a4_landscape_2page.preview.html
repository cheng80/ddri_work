강남구 따릉이 대여소별 시간대 수요 예측 실험 정리. 군집화 → 대표 15개 탐색 → 161개 운영 기준선 흐름.

  • 활용: 잔여 자전거 예측, 재배치 우선순위

피처·군집

  • 시간: 출퇴근·야간·주말·공휴일전날
  • 날씨: 기온, 습도, 강수, 강한 비 여부
  • 입지: 지하철 거리, 300m 버스정류장 수, 표고, 음식점·카페 수
  • 상호작용: 비×출퇴근, 비×야간, 강수×출퇴근, 강수×야간 (날씨 단독보다 유효)
  • 군집 5개: 업무/상업 혼합(49), 아침 도착 업무(3), 주거 도착(32), 생활·상권 혼합(61), 외곽 주거(19)

모델·실험

  • 모델: LightGBM, CatBoost / 학습 목표: 오차 최소화(RMSE), 건수형(Poisson)
  • 실험 축: 군집별 축소 피처, 정적 피처 확장, 날씨×시간대 상호작용, 군집별 분기(routing)
