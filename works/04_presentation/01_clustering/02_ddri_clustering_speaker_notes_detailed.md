# 따릉이 대여소 통합 군집화 발표 스피치 노트

## 페이지 1. 표지 및 발표 목적

### 실제 스피치

“이번 발표에서는 강남구 따릉이 대여소를 통합 군집화한 결과를 설명드리겠습니다.  
핵심은 기초 이용 패턴 분석 위에 반납 시간대와 순유입, 교통 접근성을 결합해 대여소의 지구 역할을 해석한 것입니다.  
이 군집 결과는 단순 분류에 그치지 않고, 이후 날씨나 생활인구, 고저차 같은 환경 피처를 선별해서 대여소별 수요 예측으로 연결하는 중간 기준이 됩니다.”

### 유의점

- 첫 페이지에서 “군집 자체가 목적이 아니라 예측 설계의 중간 단계”라는 점을 먼저 고정한다.
- 내부 단계보다 “통합 군집화”라는 표현을 중심으로 간다.

## 페이지 2. 분석 타겟과 피처 구성

### 실제 스피치

“이번 군집화의 타겟은 station-level입니다.  
즉 하루 수요를 바로 예측하는 것이 아니라, 각 대여소가 업무·상업형인지, 주거형인지, 생활·상권 혼합형인지, 외곽형인지 같은 공간 역할을 먼저 분류하는 작업입니다.  
메인 군집화에는 반납 시간대 비율 3개, 순유입 2개, 교통 접근성 2개로 총 7개 피처를 사용했습니다.  
그리고 이후 환경 보강 실험에서는 `station_elevation_m`, 즉 대여소 표고, `elevation_diff_nearest_subway_m`, 즉 최근접 지하철 대비 고도차, `nearest_park_area_sqm`, 즉 최근접 공원 면적, `distance_naturepark_m`, 즉 도시자연공원구역 거리, `distance_river_boundary_m`, 즉 최근접 하천경계 거리까지 추가해서 해석력 보강 가능성을 확인했습니다.”

### 유의점

- 이 페이지에서 `타겟은 station-level 군집`이라는 점을 먼저 고정한다.
- 메인 7개와 환경 보강 실험 피처를 분리해서 말한다.

## 페이지 3. 통합 군집화 접근

### 실제 스피치

“이번 분석은 먼저 기본 이용 패턴으로 대여량과 시간대 구조를 확인하고, 그 위에 지구판단용 피처를 결합하는 방식으로 진행했습니다.  
핵심은 반납 시간대, 순유입, 그리고 교통 접근성을 함께 보는 것입니다.  
이렇게 하면 단순히 많이 쓰이는 대여소를 넘어서, 업무지구형인지 주거지구형인지, 상권형인지 외곽 생활권형인지까지 해석할 수 있습니다.”

### 유의점

- 내부 분석 단계를 길게 설명하지 않는다.
- 메시지는 `기초 패턴 확인 -> 지구 역할 중심 통합 군집화` 흐름이다.

## 페이지 4. 데이터 기준과 전처리

### 실제 스피치

“데이터는 2023년과 2024년을 학습 구간, 2025년을 테스트 기준 구간으로 두었습니다.  
그리고 연도별 신규·폐쇄 대여소 영향을 줄이기 위해 2023년부터 2025년까지 공통 운영된 대여소를 기준으로 맞췄습니다.  
전처리에서는 대여일시, 반납일시, 출발 대여소번호, 반납 대여소번호, 이용시간, 이용거리 중 하나라도 비어 있으면 해당 이용 건을 제거했습니다.  
이 값들이 비어 있으면 시간대 반납 비율, 순유입, 대여소별 집계, 유효 이용 판단을 할 수 없기 때문입니다.  
그리고 시간과 거리 0 이하 값도 비정상 로그로 보고 제외했습니다.  
또 동일 대여소 반납이면서 5분 이하인 즉시 반납 로그는 미사용 가능성이 높다고 보고 제거했습니다.  
참고로 전체 289만 6,795건 중 최종 제외된 건은 23만 2,857건, 약 8.04%였습니다.  
그런데 이걸 전부 결측치 제거로 보면 안 되고, 결측 때문에 제거된 행은 0건이었습니다.  
실제 제외 대부분은 이상치와 분석 대상 제외에서 발생했습니다.  
이용시간이나 이용거리 0 이하 로그가 13만 8,057건, 동일 대여소 5분 이하 즉시 반납이 2만 2,394건, 그리고 공통 기준 밖 대여소가 7만 2,406건이었습니다.”  

### 유의점

- 전처리 설명은 짧고 명확하게.
- self-return 전체 제거가 아니라 `5분 이하 즉시 반납만 제거`임을 꼭 말한다.

## 페이지 5. 최종 군집화 입력과 원천

### 실제 스피치

“이번 군집화의 핵심 입력은 7개입니다.  
첫째, 07~10시, 11~14시, 17~20시 반납 비율로 도착 시간대 성격을 봤고,  
둘째, 아침 순유입과 저녁 순유입으로 출발 거점인지 도착 거점인지 확인했습니다.  
셋째, 최근접 지하철 거리와 300미터 내 버스정류장 수로 교통 접근성을 반영했습니다.  
중요한 점은 각 지표의 원천을 따로 관리했다는 것입니다.  
반납 시간대 비율과 순유입은 서울 열린데이터광장 공공자전거 이용정보에서 만들었고, 지하철 거리는 서울 열린데이터광장 서울시 역사마스터 정보, 버스정류장 수는 서울 열린데이터광장 서울시 버스정류소 위치정보에서 계산했습니다.”  

### 유의점

- 이 페이지에서 “지표 원천 출처”를 꼭 짚는다.
- 발표 중 질문이 나오면 원천 파일명까지 바로 말할 수 있어야 한다.

## 페이지 6. 07~10시 반납 근거

### 실제 스피치

“먼저 아침 07~10시 반납 근거입니다.  
이 페이지는 실제 HTML 지도를 캡처한 화면과 상위 반납 대여소 차트를 같이 둬서, 업무 도착형 흐름을 동시에 보게 만들었습니다.  
아침 반납이 강한 대여소는 포스코사거리, 수서역 5번출구, 매봉역 3번출구앞처럼 출근 도착 거점 성격이 분명한 지점들입니다.  
즉 이 시간대는 업무·상업지구 후보를 설명하는 가장 직접적인 근거입니다.”

### 유의점

- 지도와 차트를 같이 보면서 “업무 도착축”을 짧게 짚는다.
- 여기서는 여전히 “가설”이지 최종 용도지역 확정이 아님을 유지한다.

## 페이지 7. 11~14시 반납 근거

### 실제 스피치

“다음은 11~14시 반납 근거입니다.  
점심 시간대는 아침보다 분포가 넓어지고, 수서역 5번출구, 매봉역 3번출구앞, 선릉역3번출구처럼 업무지와 생활·상권이 섞인 지점들이 함께 나타납니다.  
그래서 이 페이지는 점심 상권형과 생활·상권 혼합형을 해석하는 근거가 됩니다.  
즉 점심 시간대 반납은 단순한 출근 도착이 아니라 업무지 내부 이동과 생활권 이동을 같이 보여줍니다.”

### 유의점

- 아침보다 분포가 넓어진다는 점을 말한다.
- 점심 시간은 상권형과 혼합형 해석의 근거라고 연결한다.

## 페이지 8. 17~20시 반납 근거

### 실제 스피치

“세 번째는 17~20시 반납 근거입니다.  
저녁에는 압구정역 2번 출구, 매봉역 3번출구앞, 자곡사거리처럼 주거 귀가 흐름과 연결되는 대여소가 상위로 올라옵니다.  
지도에서도 외곽과 주거 밀집 축에서 반납 집중이 커지는 걸 볼 수 있습니다.  
그래서 이 시간대는 주거 도착형 군집을 설명하는 핵심 근거입니다.”

### 유의점

- 저녁 반납은 귀가 흐름이라는 점을 분명히 한다.
- 외곽과 주거 밀집 축을 같이 짚는다.

## 페이지 9. 군집 수 선택 결과

### 실제 스피치

“군집 수는 이번 목적상 너무 작은 값은 피하고자 했습니다.  
지구 역할을 충분히 분리하려면 최소 5개 이상은 필요하다고 보고 `k=5~7`을 탐색했습니다.  
그 결과 silhouette score는 `k=5`에서 가장 높았고, 이번 통합 군집화의 최종 군집 수는 5개로 선택했습니다.  
즉 여기서 표에 보이는 5, 6, 7은 군집 번호가 아니라 군집 개수 후보이고, 실제 메인 군집 라벨은 Cluster 0부터 Cluster 4까지입니다.”

### 유의점

- `k=5 이상`을 본 이유를 먼저 설명한다.
- 사용자가 왜 2개나 3개가 아니라 5개인지 질문할 때 대비한다.

## 페이지 10. 군집 결과와 분포

### 실제 스피치

“최종적으로 5개 군집이 나왔고, 이 군집들은 단순 이용량보다는 도착 시간대와 순유입 구조 차이로 구분됩니다.  
예를 들어 Cluster 1은 아침 도착 비율이 압도적으로 높고 아침 순유입도 매우 커서 아침 도착 업무 집중형으로 해석할 수 있습니다.  
반면 Cluster 2와 Cluster 4는 저녁 도착 비율이 높아서 주거형 해석이 더 타당합니다.  
Cluster 3은 점심 비율도 상대적으로 높아 생활·상권 혼합형으로 볼 수 있습니다.”

### 유의점

- 군집 번호를 모두 외우려 하지 말고, `업무형 / 주거형 / 혼합형 / 외곽형` 중심으로 말한다.
- PCA 그림은 실제 분리 여부, 막대차트는 무엇이 다른지 설명하는 역할로 나눈다.

## 페이지 11. 군집 프로파일

### 실제 스피치

“이 히트맵은 군집별 평균 특성을 다시 요약한 페이지입니다.  
특히 위 히트맵은 먼저 `어떤 피처가 어느 군집에서 상대적으로 높은지`를 한 장으로 요약해 주고, 아래 사분면 차트는 그 관계를 더 설명 가능하게 풀어 주는 역할을 합니다.  
즉 히트맵은 전체 패턴 요약, 사분면 차트는 군집별 위치 관계 설명이라고 나눠서 보시면 됩니다.  

먼저 왼쪽 사분면 차트는 `07~10시 반납 비율`을 가로축, `17~20시 반납 비율`을 세로축으로 둔 그림입니다.  
점선은 전체 평균이고, 각 군집의 중심점이 어느 쪽으로 치우치는지로 해석합니다.  
오른쪽 아래로 갈수록 아침 반납 비율은 평균보다 높고 저녁 반납 비율은 평균보다 낮아서 `아침 도착형` 성격이 강합니다.  
반대로 왼쪽 위나 위쪽으로 갈수록 아침보다 저녁 반납 비율이 강해져 `저녁 도착형`, 즉 주거지 귀가 흐름 해석이 강해집니다.  

실제 수치로 보면 Cluster 1은 아침 반납 비율이 0.486으로 가장 높고 저녁 반납 비율은 0.161 수준이라 가장 전형적인 `아침 도착 업무 집중형`입니다.  
반면 Cluster 2는 아침 반납 비율이 0.113으로 낮고 저녁 반납 비율이 0.383으로 가장 높아서 가장 뚜렷한 `저녁 도착형`, 즉 주거 도착형입니다.  
Cluster 4도 저녁 반납 비율이 0.312로 높은 편이고 외곽 입지까지 겹쳐 외곽 주거형으로 읽습니다.  
Cluster 3은 저녁 반납도 높지만 점심 비율이 0.198로 상대적으로 높아서, 단순 주거형보다는 생활·상권 혼합형으로 해석하는 것이 더 자연스럽습니다.  

오른쪽 사분면 차트는 `아침 순유입`을 가로축, `저녁 순유입`을 세로축으로 둔 그림입니다.  
오른쪽이면 아침에 자전거가 들어오는 대여소, 왼쪽이면 아침에 자전거가 빠져나가는 대여소입니다.  
위쪽이면 저녁에도 순유입이 있고, 아래쪽이면 저녁에는 순유출이라는 뜻입니다.  
그래서 이 그림은 단순 시간대 비율이 아니라 `언제 자전거가 모이고 언제 빠지는가`를 보여 줍니다.  

Cluster 1은 아침 순유입이 매우 크고 저녁에는 순유출로 돌아섭니다.  
즉 출근 시간에는 자전거가 몰리고, 퇴근 시간에는 다시 빠져나가는 업무 중심축이라는 뜻입니다.  
반대로 Cluster 2와 Cluster 4는 아침에는 순유출, 저녁에는 순유입 패턴을 보여서 주거지 이동 흐름과 더 잘 맞습니다.  
Cluster 0은 아침 반납 비율과 순유입이 모두 중간 이상이라 업무/상업 혼합형으로, Cluster 3은 점심 비율과 저녁 비율이 함께 살아 있어 생활·상권 혼합형으로 구분할 수 있습니다.  

즉 이 페이지의 핵심은, 군집 차이가 단순 대여량 크기 차이가 아니라 `도착 시간대 비중`과 `순유입 방향`의 조합에서 나온다는 점을 데이터로 보여 주는 데 있습니다.”

### 유의점

- 히트맵은 모든 셀을 읽지 말고, `상대적으로 높은 축이 무엇인지`만 짚는다.
- 왼쪽 차트는 `아침 반납 vs 저녁 반납`, 오른쪽 차트는 `아침 순유입 vs 저녁 순유입`이라는 축 의미를 먼저 설명한다.
- `Cluster 1은 아침 도착형`, `Cluster 2와 4는 저녁 도착형`, `Cluster 3은 혼합형`처럼 군집별 위치를 수치와 함께 짚는다.
- 결론은 반드시 `도착 시간대와 순유입 방향의 조합이 군집을 만든다`로 닫는다.

## 페이지 12. 군집 대표 대여소와 지도

### 실제 스피치

“이 페이지는 배경 지도 위에 군집 마커와 범례를 함께 둔 정적 지도입니다.  
그래서 강남구 안에서 어떤 군집이 어디에 몰려 있는지, 외곽 주거형이 실제로 남동부 외곽에 위치하는지를 한 장에서 확인할 수 있습니다.  
그리고 아래 대표 대여소를 보면 군집 해석이 더 직관적입니다.  
업무/상업 혼합형에는 SB타워 앞, 역삼지하보도 7번출구 앞 같은 지점이 들어가고,  
아침 도착 업무 집중형에는 수서역 5번출구와 포스코사거리 같은 강한 허브가 들어갑니다.  
반면 외곽 주거형에는 세곡동이나 대형 녹지 인접 지역이 더 많이 나타납니다.”

### 유의점

- 지도에서는 군집의 공간 분포와 외곽 위치를 먼저 짚는다.
- 대표 대여소는 2~3개씩만 짧게 읽는다.

## 페이지 13. 환경 보강 실험

### 실제 스피치

“여기서 한 단계 더 나아가 환경 보강 실험도 수행했습니다.  
추가한 지표는 대여소 표고, 최근접 지하철 대비 고도차, 최근접 공원 면적, 도시자연공원구역 거리, 최근접 하천경계 거리입니다.  
원천도 모두 별도로 관리했습니다.  
표고는 Open-Meteo Elevation API, 공원 규모는 강남구 공원 정보 공공데이터, 도시자연공원구역 거리는 서울 열린데이터광장 도시자연공원구역 데이터, 하천경계 거리는 브이월드/국토정보지리원의 연속수치지형도 하천경계 데이터를 사용했습니다.  
도시자연공원구역 공식 원천은 `https://data.seoul.go.kr/dataList/OA-21135/S/1/datasetView.do`이고, 하천경계 공식 원천은 `https://www.vworld.kr/dtmk/dtmk_ntads_s002.do?svcCde=MK&dsId=30248`입니다.  
환경 보강 실험은 별도 탐색에서 `k=6`이 가장 높았기 때문에 이쪽은 실험적으로 Cluster 0부터 Cluster 5까지 나올 수 있습니다.  
결과적으로 외곽 주거형, 녹지 인접형 해석은 더 선명해졌지만, silhouette는 기본 통합 군집화보다 낮아졌습니다.  
그래서 이 결과는 메인 군집을 교체하기보다 해석을 보강하는 근거로 사용했습니다.”

### 유의점

- 이 페이지에서 “분리도는 낮아졌지만 해석은 강화됐다”는 균형을 유지한다.
- 환경 보강 결과를 메인 군집 결과보다 앞세우지 않는다.

## 페이지 14. 결론과 다음 단계

### 실제 스피치

“정리하면, 이번 통합 군집화는 기초 이용 패턴 분석에 지구판단 피처를 더해 대여소의 공간 역할을 설명하도록 구성했습니다.  
최종 발표 메인 결과는 기본 통합 군집화 `k=5`를 사용하고, 환경 보강 실험은 외곽성·녹지성·수변성 해석을 보강하는 결과로 제시합니다.  
이후에는 군집별 특성에 맞춰 날씨, 생활인구, 고저차 같은 환경 피처를 선별하고, 최종적으로 station-day 수요 예측 모델을 만들어 RMSE, MAE, R² 기준으로 평가하겠습니다.”  

### 유의점

- 마지막 문장은 반드시 `예측 모델 연결`로 닫는다.
- `RMSE`, `MAE`, `R²`를 직접 언급해 발표 전체 목적과 연결한다.
