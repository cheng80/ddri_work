# 따릉이 프로젝트 현재 구조 및 데이터 인벤토리

작성일: 2026-03-11  
최종 갱신일: 2026-03-14  
범위: `3조 공유폴더`, `works`

## 1. 이 문서의 현재 역할

이 문서는 현재 정본 구조 안내문이 아니라, 초기에 확인한 원천 데이터와 스키마를 보관하는 `인벤토리 보관 문서`다.

현재 읽기 기준 구조는 `archive_docs/05_ddri_workspace_structure_guide.md`와 각 파트 `README.md`를 우선 사용한다.

이 문서는 아래 상황에서만 다시 열면 된다.

- 원천 데이터 종류를 거칠게 다시 확인할 때
- 초기 스키마 확인 흔적이 필요할 때
- 과거 시점의 데이터 범위와 구조를 추적할 때

## 2. 원천 데이터 범주 요약

### 따릉이 이용 이력

- `2023` 월별 CSV `12개`
- `2024` 월별 CSV `12개`
- `2025` 월별 CSV `12개`

### 대여소 기준 정보

- 강남구 대여소 정보 CSV `3개`
- 연도 기준: `2023`, `2024`, `2025`

### 날씨 및 보조 환경 데이터

- 강남구 날씨 CSV
- 대기 자료 XLS 묶음
- 공원 정보
- 지하철 정보
- 버스 정류소 위치정보

### 생활인구

- 행정동별 생활인구 월별 CSV 묶음

## 3. 초기에 확인한 주요 스키마 메모

### 따릉이 이용 이력

- 실제 대여 이력 컬럼은 한글 컬럼명 기준으로 읽어야 한다
- 예:
  - `대여일시`
  - `대여 대여소번호`
  - `반납일시`
  - `반납대여소번호`

### 날씨 데이터

- 주요 컬럼:
  - `datetime`
  - `temperature`
  - `humidity`
  - `precipitation`
  - `wind_speed`

### 교통 데이터

- 지하철/버스 원천은 `cp949` 인코딩 이슈가 있을 수 있다
- 버스 좌표 컬럼은 일반적인 `위도/경도` 대신 `CRDNT_X`, `CRDNT_Y`일 수 있다

### 생활인구 데이터

- `기준일ID`
- `시간대구분`
- `행정동코드`
- `총생활인구수`

## 4. 현재 기준으로 남길 핵심 메모

- 최신 군집화 결과 해석은 `works/01_clustering/08_integrated/final` 기준으로 한다
- 학습/테스트 연도는 `2023~2024 / 2025`로 고정한다
- 파일명, 노트북명, 산출물명은 `ddri_*` 규칙을 유지한다
- 원천 데이터 상세 목록보다 `현재 어떤 정본 경로를 읽어야 하는지`가 더 중요하므로, 구조 안내는 이 문서가 아니라 `archive_docs/05_ddri_workspace_structure_guide.md`에서 관리한다
