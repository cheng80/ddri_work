# 02 Data Collection Archive

이 폴더는 과거 외부 데이터 수집 작업물을 보관하는 archive 경로다.

현재 학습과 해석에 직접 쓰는 값은 이미 생성된 CSV나 학습 데이터셋에 반영되어 있으므로, 이 폴더는 메인 읽기 경로가 아니다.

## 구성

- `01_calendar`
  - 공휴일 API 수집 노트북, 스크립트, 원시/일별 캘린더 CSV
- `02_weather`
  - Open-Meteo 수집 노트북, 스크립트, 보정용 날씨 CSV

## 현재 해석

- 재수집 절차를 다시 봐야 할 때만 이 폴더를 연다
- 일반적인 프로젝트 읽기 흐름에서는 `works/00_overview`, `works/05_prediction_long`, `works/06_prediction_long_full`을 우선 본다
