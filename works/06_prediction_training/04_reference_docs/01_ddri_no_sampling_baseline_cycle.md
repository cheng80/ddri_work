# DDRI 무샘플링 기준선 1사이클

작성일: 2026-03-17

## 1. 목적

샘플링 없이 정본 전체를 사용해 기준선 성능을 먼저 확보한다.

이 기준선은 이후:

- 가중치 방식
- 일단위 대표일 추출
- 기타 삭제형 샘플링

과 비교하는 기준이 된다.

## 2. 적용 범위

- 입력: `canonical_data`
- 샘플링: 없음
- 행 삭제: 없음
- 시간 분할:
  - train `2023`
  - validation `2024`
  - test `2025`

## 3. 1사이클 순서

1. 정본 로드
2. 전처리
3. `2023` 학습
4. `2024` 검증
5. 우세 모델 선택
6. `2023+2024` 재학습
7. `2025` 테스트

## 4. 전처리 원칙

- 정본의 구조적 결측은 정본 단계에서 임의 보간하지 않는다.
- 모델링 단계에서만 `missing_flag`를 추가한다.
- 샘플 행은 제거하지 않는다.
- 샘플 가중치도 기본선에서는 사용하지 않는다.

## 5. 저장 지표

검증과 테스트 모두 아래 지표를 저장한다.

- `RMSE`
- `MAE`
- `R²`

## 6. 현재 해석

지금 단계에서 무샘플링 1사이클을 먼저 수행해야 하는 이유는 아래와 같다.

- 이후 샘플링이 실제로 도움이 되는지 비교 기준이 필요하다.
- 샘플링 없이도 충분히 성능이 나오는지 먼저 확인해야 한다.
- 삭제형 샘플링은 분포 왜곡 위험이 있어 기준선 없이 바로 적용하면 판단이 어렵다.
