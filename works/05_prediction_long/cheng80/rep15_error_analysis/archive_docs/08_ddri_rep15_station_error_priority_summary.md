# DDRI 대표 15개 스테이션 오류 우선순위 요약

작성일: 2026-03-14  
목적: 대표 15개 `station-hour` 실험에서 상위 오류 스테이션을 우선 해석하고, 전체 161개 상위 오류와 겹치는 핵심 스테이션을 함께 확인한다.

## 1. 결론 요약

현재 바로 해석해야 할 우선 대상은 대표 15개 기준 상위 오류 스테이션이다.

특히 아래 5개를 먼저 보는 것이 적절하다.

1. `2377` 수서역 5번출구
2. `2348` 포스코사거리(기업은행)
3. `4917` 일원에코파크 주차장
4. `2328` 르네상스 호텔 사거리 역삼지하보도 7번출구 앞
5. `2359` 국립국악중,고교 정문 맞은편

이유:

- 대표 15개는 이미 군집 설명과 사례 해석 기반이 갖춰져 있다.
- 전체 161개 상위 오류 Top20와 직접 겹치는 대표 스테이션은 `2377`, `2348` 두 곳뿐이다.
- 따라서 대표 15개에서 먼저 오류 해석 프레임을 만들고, 그중 `2377`, `2348`를 전체 161개 확장 해석의 연결 고리로 쓰는 것이 합리적이다.

## 3. 군집별 해석 출발점

- `아침 도착 업무 집중형`
  - 평균 RMSE가 가장 높다
  - `2377`, `2348`가 동시에 대표 15개와 전체 161개 상위 오류에 걸쳐 있어 최우선 분석 군집이다
- `주거 도착형`
  - `4917`이 대표 15개 내부에서는 강한 오류 스테이션이다
- `업무/상업 혼합형`
  - `2328`, `4902` 중심으로 해석을 시작할 수 있다
- `외곽 주거형`
  - `2359`, `3643`를 보면 된다
- `생활권 혼합형`
  - 절대 오류는 상대적으로 낮지만 `2320`이 대표 군집 사례로 적절하다
