# DDRI 대표 15개 상위 5개 오류 스테이션 시간대 패턴 요약

작성일: 2026-03-14  
목적: 대표 15개 상위 오류 5개 스테이션의 시간대별 실제값-예측값 차이를 비교해, 다음 군집별 subset 실험의 우선 해석 포인트를 정리한다.

## 1. 대상 스테이션

이번 확장 분석 대상은 아래 5개다.

1. `2377` 수서역 5번출구
2. `2348` 포스코사거리(기업은행)
3. `4917` 일원에코파크 주차장
4. `2328` 르네상스 호텔 사거리 역삼지하보도 7번출구 앞
5. `2359` 국립국악중,고교 정문 맞은편

## 2. 전체 해석

- `2377`, `2348`는 여전히 `16~19시` 저녁 피크 시간대가 핵심 오차 구간이다.
- `4917`은 저녁보다 `05~08시` 아침 시간대 과소예측이 더 뚜렷하다.
- `2328`은 절대 오차 수준은 앞의 세 스테이션보다 낮지만 `17~18시`와 `08시`, `11시`에서 확인 필요 구간이 보인다.
- `2359`는 `08시`, `16~18시`, `20시`에 오차가 모이며, 외곽 주거형의 생활권 이동 시간대를 같이 봐야 한다.
- 즉 대표 15개 상위 5개는 하나의 공통 패턴이 아니라, `저녁 업무권 피크형`과 `아침 주거/외곽 이동형`이 섞여 있다.

## 3. 스테이션별 요약

  - 평균 예측값: `1.9304`

  - RMSE: `1.7168`
  - 평균 편향: `-0.1078`
- 상위 오차 시간대:
  - `18시`
  - `19시`
  - `17시`
  - `20시`
  - `16시`
- 해석:
  - 저녁 피크 시간대 변동이 가장 큰 병목이다.
  - 전반적으로 예측 방향이 한쪽으로 고정되기보다, 피크 시간 수요 크기를 안정적으로 따라가지 못한다.
