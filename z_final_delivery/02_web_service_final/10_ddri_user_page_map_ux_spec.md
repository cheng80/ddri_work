# DDRI 사용자 페이지 – 지도 UX 설계

작성일: 2026-03-17  
목적: 지도·리스트 레이아웃 개선. 카드당 개별 지도 제거, **단일 지도 + 원형 반경 + 마커** 구조로 정리.

---

## 1. 기존 이슈

- Stitch 디자인: 대여소 카드마다 **개별 지도 썸네일** → 불필요한 반복, 화면 낭비
- 개선 방향: **지도 1개** + **원형 반경** + **마커** + **카드 리스트**

---

## 2. 지도 영역 설계

### 2.1 단일 지도 구성

| 요소 | 설명 |
|------|------|
| **중심점** | 사용자 위치 (현 위치 또는 주소 검색 결과) |
| **원형 반경** | 반경 선택 (예: 300m / 500m / 1km) – 반경 내 대여소만 표시 |
| **마커** | 대여소 위치, 색상으로 대여가능/보통/부족 구분 |
| **클릭/탭** | 마커 선택 시 해당 카드로 스크롤 또는 상세 툴팁 |

### 2.2 반경 옵션 (예시)

- 300m
- 500m (기본)
- 1km

---

## 3. 모바일 레이아웃

### 3.1 권장: 지도 상단 + 카드 리스트 하단

```
┌─────────────────────────────────────┐
│  [현 위치] [주소 검색]               │
│  현재 위치: 서울시 강남구 역삼동...    │
├─────────────────────────────────────┤
│  ┌─────────────────────────────┐   │
│  │                             │   │
│  │   단일 지도 (원형 반경)       │   │
│  │   • 사용자 위치 마커          │   │
│  │   • 대여소 마커들             │   │
│  │   • 반경 500m 원             │   │
│  │                             │   │
│  └─────────────────────────────┘   │
│  [300m] [500m] [1km]  ← 반경 선택   │
├─────────────────────────────────────┤
│  대여소 목록 (카드형, 세로 스크롤)   │
│  ┌─────────────────────────────┐   │
│  │ 강남역 7번출구  150m  대여가능 │   │
│  └─────────────────────────────┘   │
│  ┌─────────────────────────────┐   │
│  │ 신논현역 5번출구 340m  보통   │   │
│  └─────────────────────────────┘   │
│  ...                                │
└─────────────────────────────────────┘
```

### 3.2 대안: 지도/리스트 탭 전환

- **지도 탭**: 전체 화면 지도 + 마커
- **리스트 탭**: 카드 리스트만
- 장점: 각 뷰에 화면을 더 쓸 수 있음  
- 단점: 한 화면에서 동시에 보지 못함

### 3.3 권장안

- **기본**: 지도 상단 + 리스트 하단 (스크롤로 연속 탐색)
- 지도 높이: 화면 높이의 35~40% (약 200~250dp)

---

## 4. 데스크탑 레이아웃

### 4.1 권장: 2분할 (지도 좌측 + 리스트 우측)

```
┌──────────────────────────────────────────────────────────────────┐
│  [현 위치] [주소 검색]   현재 위치: 서울시 강남구 역삼동...         │
├────────────────────────────┬─────────────────────────────────────┤
│                            │  대여소 목록 (카드형, 스크롤)        │
│   단일 지도                 │  ┌─────────────────────────────┐   │
│   (원형 반경, 마커)         │  │ 강남역 7번출구  150m  대여가능 │   │
│                            │  └─────────────────────────────┘   │
│   • 사용자 위치             │  ┌─────────────────────────────┐   │
│   • 대여소 마커             │  │ 신논현역 5번출구 340m  보통   │   │
│   • 반경 원                 │  └─────────────────────────────┘   │
│                            │  ...                                │
│   [300m] [500m] [1km]      │                                     │
│                            │                                     │
└────────────────────────────┴─────────────────────────────────────┘
```

- **지도 비율**: 40~50% (예: 5:5 또는 4:6)
- **리스트**: 세로 스크롤, 카드 1열 또는 2열

### 4.2 대안: 지도 상단 전체폭 + 리스트 하단

```
┌──────────────────────────────────────────────────────────────────┐
│  [현 위치] [주소 검색]   현재 위치: 서울시 강남구 역삼동...         │
├──────────────────────────────────────────────────────────────────┤
│                                                                  │
│   단일 지도 (전체 폭, 높이 300~400px)                             │
│   원형 반경 + 마커                                                │
│                                                                  │
├──────────────────────────────────────────────────────────────────┤
│  대여소 목록 (그리드 2~3열)                                       │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐                          │
│  │ 카드 1   │ │ 카드 2   │ │ 카드 3   │  ...                      │
│  └──────────┘ └──────────┘ └──────────┘                          │
└──────────────────────────────────────────────────────────────────┘
```

- 지도가 넓어 위치 파악에 유리
- 리스트는 그리드로 여러 카드 동시 노출

### 4.3 권장안

- **데스크탑**: 2분할 (지도 좌측 + 리스트 우측)
- 지도와 리스트를 동시에 보며, 마커 클릭 시 해당 카드 하이라이트/스크롤 연동

---

## 5. 반응형 요약

| 디바이스 | 지도 | 리스트 | 비고 |
|----------|------|--------|------|
| **모바일** | 상단, 높이 35~40% | 하단 세로 카드 | 지도 1개, 원형 반경, 마커 |
| **태블릿** | 상단 또는 좌측 40% | 하단/우측 | 2분할 또는 스택 |
| **데스크탑** | 좌측 40~50% | 우측 카드(1~2열) | 2분할, 마커↔카드 연동 |

---

## 6. 마커·카드 연동

- **지도 마커 클릭** → 리스트에서 해당 대여소 카드로 스크롤 + 하이라이트
- **리스트 카드 클릭** → 지도에서 해당 마커로 줌/포커스
- (선택) 마커 호버 시 툴팁: 대여소명, 거리, 자전거 수

---

## 7. 구현 시 참고

- **지도 라이브러리**: Flutter Web → `google_maps_flutter_web`, `flutter_map`(OpenStreetMap), 또는 Kakao Map API
- **원형 반경**: 지도 위 Circle 오버레이 (반경 m → 픽셀 변환)
- **마커 색상**: 대여가능(녹색), 보통(주황), 부족(빨강)
