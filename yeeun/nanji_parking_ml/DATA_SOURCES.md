# 난지 외부데이터 소스 정리

작성일: 2026-04-02

## 1. 캠핑장 마스터 데이터

### 1-1. 마포구 캠핑장 현황 파일데이터

- 링크: https://www.data.go.kr/data/15126656/fileData.do
- 제공기관: 서울특별시 마포구
- 포털 표기 파일명: `서울특별시 마포구_캠핑장 현황_20250403`
- 포털 설명 기준 포함 정보:
  - 캠핑장 명칭
  - 도로명 주소
  - 행정동
  - 영업시작일자
  - 캠핑사이트 수
- 활용 포인트:
  - 난지 캠핑장 기준 정보 정리
  - 주소와 사이트 수 기반의 마스터 테이블 생성

## 2. 날씨 데이터

### 2-1. 기상자료개방포털 Open API

- 링크: https://data.kma.go.kr/api/selectApiDetail.do?pgmNo=42
- 용도:
  - 시간대별 기온, 강수, 습도, 풍속 feature 생성

### 2-2. Open-Meteo Historical Weather API

- 링크: https://open-meteo.com/
- 용도:
  - 빠른 프로토타입용 백업 소스
  - 좌표 기반 시간 단위 날씨 결합

## 3. 달력/공휴일 데이터

### 3-1. 공공데이터포털 특일 정보 API

- 포털: https://www.data.go.kr
- 추천 키워드:
  - `특일 정보`
  - `공휴일 API`
- 용도:
  - `is_holiday`
  - 명절/대체공휴일 플래그

## 4. 공연정보 데이터

### 4-1. KOPIS 공연목록 API

- 링크: https://www.data.go.kr/data/15097805/openapi.do
- 제공기관: 예술경영지원센터
- 주요 필드:
  - 공연명
  - 공연시설명
  - 공연시작일자
  - 공연종료일자
  - 장르
- 활용 포인트:
  - 날짜 기반 `event_flag`
  - 한강공원 인근 문화행사 수요 반영

### 4-2. KOPIS 공연시설목록 API

- 링크: https://www.data.go.kr/data/15097806/openapi.do
- 제공기관: 예술경영지원센터
- 주요 필드:
  - 공연시설명
  - 지역 시도 코드
  - 지역 구군 코드
  - 시설 특성 코드
- 활용 포인트:
  - 서울 공연장 필터링
  - 마포구 또는 한강 접근권 공연장 후보 추리기

## 5. 참고 논문

### 5-1. 한강공원 이용 패턴 연구

- 링크: https://www.jkila.org/archive/view_article?pid=jkila-53-3-31
- 논문명: `유동인구 데이터를 활용한 한강공원의 이용 패턴과 특성 연구`
- 확인일 기준 핵심 메모:
  - 2023년 KT 유동인구 데이터를 사용
  - 한강공원 11개 지구 분석
  - 난지는 공원별 시간대 분석 요약에서 `16시`가 첨두시각으로 제시됨
- 활용 포인트:
  - 공연/행사 플래그가 특히 늦은 오후 수요에 미치는 영향 가설 설정
  - 난지와 인접 공원 비교 시 참고 근거로 사용

## 6. 정제 기준

`prepare_external_datasets.py`는 아래 기준으로 정제합니다.

- `2023`, `2024`, `2025` 연도만 유지
- 핵심 날짜/식별 컬럼 결측 행 제거
- 숫자형 컬럼에 대해 IQR 기준 이상치 제거
- 정제 결과 요약을 `cleaning_summary.csv`와 `cleaning_summary.json`에 저장
