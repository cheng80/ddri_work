# 난지 한강공원 주차장 예측 데이터 소스 정리

작성일: 2026-04-02

이 문서는 난지 한강공원 주차장 빈자리 예측 프로젝트에서 사용할 외부 데이터 소스를 정리한 팀 공유용 문서입니다.

## 우선순위

1. 주차장 원천 데이터
2. 날씨 데이터
3. 공휴일/캘린더 데이터
4. 난지 특화 보강 데이터
5. 필요 시 크롤링

## 1. 서울 열린데이터광장

### 1-1. 한강공원 주차장 정보

- 목적: 주차장 마스터 데이터 확보
- 추천 용도:
  - `parking_lot_id`
  - 주차장명
  - 위치
  - 총 주차면수
  - 운영 시간
  - 운영 기관
- 링크:
  - https://data.seoul.go.kr/dataList/OA-21083/A/1/datasetView.do
- 2026-04-02 확인 메모:
  - 서울 열린데이터광장 검색 결과 기준 `한강공원 주차장 정보` 데이터셋 존재 확인
  - 설명상 지구별, 주차장명, 위치정보, 면수, 운영시간, 요금 등의 항목 제공
  - 검색 결과 스니펫 기준 데이터 갱신일은 `2025-10-20`로 표시됨

### 1-2. 서울시 공영주차장 안내 정보

- 목적: 구조가 유사한 주차장 데이터셋 후보 확보
- 추천 용도:
  - 한강공원 외 일반 공영주차장 비교 데이터
  - 컬럼 구조 참조
- 작업 메모:
  - 실제 프로젝트 착수 시 서울 열린데이터광장에서 아래 키워드로 추가 검색
  - 검색 키워드:
    - `공영주차장 이용정보`
    - `공영주차장 실시간`
    - `주차장 입출차`

### 1-3. 한강공원 관련 추가 검색 키워드

- `한강공원 주차장`
- `한강공원 주차장 이용`
- `한강사업본부 주차`
- `난지 한강공원 주차장`

## 2. 기상청 API

- 목적: 시간대별 날씨 feature 결합
- 링크:
  - https://data.kma.go.kr/api/selectApiDetail.do?pgmNo=42
- 2026-04-02 확인 메모:
  - 기상자료개방포털의 Open API 안내 페이지 존재 확인
  - 안내에 따르면 실제 활용 신청은 공공데이터포털(data.go.kr) 게이트웨이 방식 기준으로 진행

### 추천 feature

- `weather_temp_c`
- `weather_precip_mm`
- `weather_humidity`
- `weather_wind_speed`

### 결합 grain

- 예측 테이블이 시간 단위면 날씨도 시간 단위 결합
- 예측 테이블이 일 단위면 평균/최대/누적 기준으로 집계 후 결합

### 실무 메모

- 기상청은 공식성이 가장 높아서 팀 발표나 보고서에 유리함
- API 키 발급과 호출 제한을 같이 확인해야 함

## 3. Open-Meteo

- 목적: 빠른 프로토타이핑 또는 기상청 대체/보완
- 링크:
  - https://open-meteo.com/
- 2026-04-02 확인 메모:
  - 공식 사이트 기준 시간별 historical weather API 문서 검색 가능

### 추천 사용 시점

- 기상청 API 키 발급 전 빠른 테스트가 필요할 때
- 좌표 기반 시계열 데이터를 간단히 붙이고 싶을 때

### 주의

- 최종 발표/공식 서비스 문서에는 기상청 우선 사용이 설명상 더 안정적

## 4. 공휴일 데이터

### 4-1. 공공데이터포털 특일 정보 API

- 목적: 공휴일/대체공휴일/명절 전후 효과 반영
- 추천 후보:
  - 한국천문연구원 특일 정보 API
- 검색 진입 링크:
  - https://www.data.go.kr/
- 검색 키워드:
  - `특일 정보`
  - `공휴일 API`
  - `한국천문연구원 공휴일`

### 4-2. 직접 생성 가능한 캘린더 feature

- `hour`
- `day_of_week`
- `is_weekend`
- `month`
- `is_month_start`
- `is_month_end`

### 실무 추천

- 최소 구현은 pandas로 가능
- 서비스 품질을 높이려면 공휴일 API를 같이 붙이는 것이 좋음

## 5. 난지 특화 데이터

### 5-1. 캠핑장/BBQ/시설 이용량

- 목적: 난지 한강공원만의 수요 급증 요인 반영
- 후보:
  - 난지 캠핑장 예약 현황
  - 바비큐장 이용량
  - 한강공원 행사 일정

### 활용 방법

- `event_flag`
- `camping_high_demand_flag`
- `festival_day_flag`

### 확보 방법

- 공식 예약 페이지 또는 운영 공지 페이지 확인
- 비정형 공지만 있다면 별도 수기 정리 CSV 구축 가능

## 6. 크롤링

- 목적: 공식 API가 없을 때 보조적으로 실시간 주차 정보를 수집
- 대상 예시:
  - 한강사업본부 주차 포털
  - 실시간 잔여면 표시 웹페이지

### 주의

- robots.txt, 이용약관, 요청 빈도 확인 필요
- 공식 API가 있으면 크롤링보다 API 우선
- 팀 공유용 저장소에는 크롤링 대상 URL, 로그인 필요 여부, HTML 구조 변경 위험을 같이 기록하는 것이 좋음

1. 서울 열린데이터광장에서 한강공원 주차장 마스터 확보
2. 실제 시간별 주차 점유/빈자리 데이터 소스 확보
3. 기상청 또는 Open-Meteo로 날씨 결합
