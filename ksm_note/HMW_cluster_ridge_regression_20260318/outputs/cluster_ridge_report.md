## 1. 실험 설정
- 모델: Ridge(alpha=2.0)
- 전처리: median imputation + standardization
- 학습 가중치: sample_weight 적용
- 분할: 2023=학습, 2024=검증, 2025=테스트
- 군집별 모델에서는 `cluster` 자체를 입력 feature에서 제외
