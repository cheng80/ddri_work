작성일: 2026-03-15  
대상 군집: `외곽 주거형` (`2359`, `2392`, `3643`)

## 1. 보고서 목적

이번 보고서의 목적은 아래 3가지다.

1. 외곽 주거형 Top3 대여소 기준으로 현재 모델링 기준점(benchmark)을 명확히 정리
2. baseline 대비 feature 추가 전략이 실제로 유효한지 검증
3. 운영 가능한 최종안(모델 + feature 세트)을 확정

## 2. 데이터/평가 기준

- 입력 데이터:
  - `ksm_note/Data/ddri_prediction_long_train_2023_2024_second_round_feature_collection.csv`

## 4. Ablation (1개/2개 피처 추가) 결과

`LightGBM_RMSE` 기준으로 총 29개 조합(기본 1, 단일 7, 2개조합 21)을 검증했다.

- 기준 baseline: validation RMSE `0.786281`, test RMSE `0.716329`
- 단일 추가 최적: `+ is_weekend`
  - validation RMSE `0.785635` (기준 대비 `-0.000646`)
  - test RMSE `0.714925` (기준 대비 `-0.001404`)
- 2개 조합 최적(validation 기준): `+ distance_naturepark_m + distance_river_boundary_m`
  - validation RMSE `0.785291` (기준 대비 `-0.000990`)
  - test RMSE `0.715119` (기준 대비 `-0.001210`)

해석:

- 개선 조합은 확인되지만 폭이 매우 작다.
- 따라서 “고복잡도 커스텀”보다 “경량 커스텀”이 더 실무적이다.
