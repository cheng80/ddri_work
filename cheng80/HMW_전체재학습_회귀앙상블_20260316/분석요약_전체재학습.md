# HMW station-hour 회귀 앙상블 (train/test 직접 사용)

## 1) 왜 빨라졌는가

- `station_hour_bike_flow_train_2023_2024.csv`, `station_hour_bike_flow_test_2025.csv`를 직접 사용
- 이전처럼 전체 raw에서 weather/meta를 재머지하고 전체 피처를 재생성하는 단계 생략

## 2) 실험 설정

- 타깃: `bike_change`
- train rows: `3,193,008`
- test rows: `1,594,320`
- feature shape: train `(3193008, 27)`, test `(1594320, 27)`
