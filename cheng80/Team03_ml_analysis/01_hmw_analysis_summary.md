## 2. hmw2 – 회귀 모델 벤치마크

### 2.1 분석 목표

- **단위**: station-hour (대여소·시간)
- **학습**: 2023~2024 / **테스트**: 2025
- **타깃**: `bike_change`(시간별 변화량), `bike_count_index`(재고 지수)

### 2.2 최종 모델 성능

| 타깃 | 모델 | RMSE | MAE | R² |
|------|------|------|-----|-----|
| **bike_change** | hist_gbm_tuned | 1.0543 | 0.5870 | 0.2057 |
| **bike_count_index** | ridge | 1.6539 | 1.0867 | 1.0000 |

- **bike_change**: 다음 1시간 순증가/순감소 예측 → **실제 서비스용 타깃**
- **bike_count_index**: 대여/반납 누적 지수 → 해석용

### 2.3 모델 순위 (bike_change)

1. hist_gbm_tuned (1.054)
2. hist_gbm (1.056)
3. extra_trees (1.074)
4. random_forest (1.075)
5. ridge (1.102)

### 2.4 주요 피처 (bike_change)

- **lag**: bike_change_lag_168, bike_change_lag_24
- **시간**: hour_sin, hour, is_weekend_or_holiday, is_commute_hour
- **rolling**: bike_change_rollmean_24, bike_change_rollstd_168
- **기타**: return_count_lag_24, rental_count_lag_1

### 2.5 타깃 난이도

- 정규화 RMSE 기준 **bike_count_index**가 더 쉬움 (R²≈1)
- bike_change는 변동성이 커서 예측 난이도가 높음

## 3. hmw3 – 군집 결합·대표 스테이션 분석

### 3.1 군집 피처 반영 효과

| 타깃 | cluster 미포함 | cluster 포함 | 개선 |
|------|----------------|--------------|------|
| bike_change | RMSE 1.0543 | RMSE 1.0508 | -0.0035 |
| bike_count_index | RMSE 1.6539 | RMSE 1.1019 | -0.5520 |

- bike_change: 군집 추가 시 **소폭 개선**
- bike_count_index: 군집 추가 시 **큰 개선**

### 3.2 군집 안정성

- **161개 스테이션 중 35개**가 2023–2024 train → 2025 test에서 **군집 변경**
- **변경 비율**: 21.74%
- 군집은 고정 정답이 아니라 **보조 피처**로 해석

### 3.3 군집별 특화 모델 vs 글로벌 모델

| 군집 | 글로벌 RMSE | 특화 RMSE | 특화 모델 |
|------|-------------|-----------|-----------|
| 업무/상업 혼합형 | 1.091 | 1.082 | hist_gbm_tuned |
| 아침 도착 업무 집중형 | 1.818 | **1.724** | hist_gbm_tuned |
| 주거 도착형 | 1.193 | 1.178 | hist_gbm_tuned |
| 생활·상권 혼합형 | 1.007 | 1.008 | hist_gbm |
| 외곽 주거형 | 0.945 | **0.935** | hist_gbm_tuned |

- **아침 도착 업무형·외곽 주거형**에서 특화 모델이 더 유리
- **전체**: 글로벌 1.084 vs 특화 1.074 → 특화 모델이 소폭 우수

### 3.4 대표 스테이션 15개 (군집별 3개)

| 군집 | 대표 스테이션 예시 |
|------|-------------------|
| 업무/상업 혼합형 | 3호선 매봉역 3번출구앞, 삼호물산버스정류장, 도곡역 아카데미스위트 |
| 아침 도착 업무 집중형 | 수서역 5번출구, 포스코사거리, 주식회사 오뚜기 정문 앞 |
| 주거 도착형 | 대모산입구역 4번 출구, 자곡사거리, 압구정역 2번 출구 |
| 생활·상권 혼합형 | 선릉역3번출구, 대치역 7번출구, 은마아파트 입구 사거리 |
| 외곽 주거형 | 더시그넘하우스 앞, 래미안강남힐즈 사거리, LH강남힐스테이트아파트 |

- 선정 기준: 이용량(usage), 군집 중심 거리(center_distance), MAE 등

### 3.5 경량 피처 버전

- **전체 62개** → **21개**로 축소
- RMSE 1.0508 → 1.0534 (소폭 악화)
- 운영 단계에서는 **경량 버전** 사용 가능
